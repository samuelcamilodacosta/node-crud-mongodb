# Node CRUD MongoDB

REST API developed with Node.js, TypeScript, Express and MongoDB following a layered architecture and repository pattern.

This project demonstrates the implementation of a CRUD API with database integration, automated tests, Docker support and API documentation.

## Features

- Create records
- List records
- Find record by ID
- Update records
- Delete records
- MongoDB integration with TypeORM
- API documentation with Swagger
- Unit tests with Jest
- Dockerized environment
- TypeScript support
- ESLint and Prettier configuration
- Git hooks with Husky

## Tech Stack

- Node.js
- TypeScript
- Express
- MongoDB
- TypeORM
- Swagger
- Docker
- Husky
- ESLint
- Prettier
